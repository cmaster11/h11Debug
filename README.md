# h11Debug

An extendend debug wrapper.

## Installation

    npm install h11Debug --save
  
## Usage

    var debug = require('h11Debug')('debugID');
    
    debug('Debug message');

### Environment

Create an environment variable like this:

    export DEBUG=H11D*

## Reference

For reference about this variable: [https://github.com/visionmedia/debug]().
